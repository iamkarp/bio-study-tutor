# Homologous Chromosomes

## Summary
A **matching pair of chromosomes** in a diploid cell — one inherited from each parent. They look alike, are the same length, and carry **the same genes in the same order** at the same loci, but the **alleles** at any given locus may differ between the maternal and paternal copy. During [[processes/meiosis-i]], homologous chromosomes pair up (synapsis) and separate from each other.

Sex chromosomes X and Y are only partly homologous (they pair only over a short region).
