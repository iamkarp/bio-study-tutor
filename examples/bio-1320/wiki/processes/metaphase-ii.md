# Metaphase II

## Summary
**Duplicated chromosomes line up singly** (not in pairs) at the equator, perpendicular to the spindle. This is mechanically very similar to mitotic metaphase, but it occurs in haploid cells with one copy of each chromosome (still as sister chromatids).
