# S Phase (Synthesis Phase)

## Summary
The phase of [[processes/cell-cycle]] interphase during which **DNA replication** occurs. Each chromosome is fully copied; afterward each chromosome consists of two **[[structures/sister-chromatids]]** held at the centromere. The cell ends S phase with twice the DNA but the same chromosome count (still 2n in a diploid cell).
