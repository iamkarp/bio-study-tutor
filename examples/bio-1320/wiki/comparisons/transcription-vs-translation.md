# Transcription vs. Translation

## Comparison

| Feature | Transcription | Translation |
|---|---|---|
| **Template** | DNA | mRNA |
| **Product** | RNA (mRNA, tRNA, rRNA) | Protein (polypeptide) |
| **Enzyme / machinery** | RNA polymerase | Ribosome (rRNA + proteins; tRNAs as adaptors) |
| **Location (eukaryote)** | Nucleus | Cytoplasm (free ribosomes) or rough ER |
| **Read in units of** | Single nucleotides | Codons (3 nucleotides → 1 amino acid) |
| **Start signal** | Promoter | AUG (start codon) → methionine |
| **Stop signal** | Terminator sequence | UAA, UAG, UGA (stop codons) → release factor |
| **Direction** | Template read 3'→5'; RNA built 5'→3' | mRNA read 5'→3' |
| **Pre-product processing?** | Yes (cap, tail, splicing in eukaryotes) | Yes (folding, modification, transport) |
