# Prokaryote vs. Eukaryote DNA Replication

## Comparison

| Feature | Prokaryotes | Eukaryotes |
|---|---|---|
| **Chromosome shape** | Single large **circular** chromosome | Multiple **linear** chromosomes |
| **Extra DNA** | [[structures/plasmid|Plasmids]] (small circular) | None (organelle DNA exists but separate) |
| **Origins of replication** | **One** ORI per chromosome | **Many** ORIs per chromosome |
| **Replication bubbles** | One | Many; merge as they expand |
| **Speed** | Faster (~1000 nt/sec) | Slower (~50 nt/sec) — but parallel ORIs make up for it |
| **Telomere problem?** | No (circular DNA doesn't shorten) | Yes — solved (in some cells) by [[enzymes/telomerase]] |
| **Coupled to transcription?** | Often (no nucleus) | Decoupled (transcription in nucleus, replication in S) |
