# DNA Polymerase (general / PCR)

## Summary
Generic name for an enzyme that **synthesizes DNA from a template** using free dNTPs. In **PCR**, a heat-stable DNA polymerase like **Taq polymerase** (from *Thermus aquaticus*) is used so the enzyme isn't destroyed during the 95 °C denaturation step. In cells, multiple specialized polymerases exist (see [[enzymes/dna-polymerase-iii]], [[enzymes/dna-polymerase-i]]).
