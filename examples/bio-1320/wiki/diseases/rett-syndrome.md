# Rett Syndrome

## Summary
An **X-linked** neurodevelopmental disorder. **Lethal in males** (a boy with the mutation has only one X, so it's expressed in every cell). In girls, severity varies depending on **[[concepts/x-inactivation|X-inactivation]]** patterns — the more cells that randomly inactivate the X with the Rett allele, the milder the disease, because the unaffected X is active in those cells.
