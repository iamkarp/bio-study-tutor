# Genotype

## Summary
An organism's **two alleles for a given gene** (or set of genes). Written with capital letters for dominant alleles and lowercase for recessive (e.g. **YY, Yy, yy**). Genotype determines [[terms/phenotype|phenotype]], but two organisms with the same phenotype may have different genotypes (homozygous dominant vs. heterozygous both look dominant).
