# Start and stop codons

## Answer
- **Start codon: AUG** — codes for **methionine (Met)**. AUG marks where translation begins; the initiator tRNA carrying Met base-pairs with this codon to start protein synthesis. **Every protein begins with methionine** (although it can be removed later).
- **Stop codons: UAA, UAG, UGA** — do **not** code for any amino acid. When the ribosome encounters a stop codon, a **release factor** binds in place of a tRNA, and the finished protein is released from the ribosome.

There are 64 possible codons (4³). 61 specify amino acids; 3 are stop codons.
