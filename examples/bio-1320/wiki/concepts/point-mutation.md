# Point Mutation

## Summary
A mutation that affects **a single base pair** in a gene. Includes [[concepts/substitution-mutation|substitutions]], single-base [[concepts/insertion-mutation|insertions]], and single-base [[concepts/deletion-mutation|deletions]]. The most extreme example: [[diseases/sickle-cell-anemia|sickle cell]] arises from a single A→T substitution in the β-hemoglobin gene.
