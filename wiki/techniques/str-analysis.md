# STR Analysis

## Summary
A forensic technique that identifies individuals by analyzing **[[concepts/short-tandem-repeat|short tandem repeats]] (STRs)** at multiple genomic loci. The standard CODIS panel uses **13 STR loci**.

## Process
1. **Collect DNA** from the crime scene or sample.
2. **Amplify each STR locus** using [[techniques/pcr]] — each person inherits two copies (one per chromosome) at each locus, possibly with different repeat numbers.
3. **Run on [[techniques/gel-electrophoresis|gel electrophoresis]]** — bands position by length, which depends on repeat count.
4. **Compare profiles** — match across all 13 STRs gives a probability of ~1 in a trillion of a random match. Mismatch on even one locus excludes a suspect.

Used to identify the Golden State Killer (Joseph James DeAngelo) and many cold cases.
