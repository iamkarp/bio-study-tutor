# Transcription vs. translation

## Answer

| | **Transcription** | **Translation** |
|---|---|---|
| What's made | RNA copy of a gene | Protein |
| Template | DNA | mRNA |
| Enzyme | RNA polymerase | Ribosome (rRNA + proteins; tRNAs as adaptors) |
| Where (eukaryote) | Nucleus | Cytoplasm (at ribosomes, free or on ER) |
| Reads in units of | Nucleotides | Codons (3 nucleotides → 1 amino acid) |
| Start signal | Promoter | Start codon (AUG → methionine) |
| Stop signal | Terminator | Stop codon (UAA, UAG, or UGA) |

In short: **transcription** is DNA → RNA inside the nucleus; **translation** is mRNA → protein at a ribosome.
