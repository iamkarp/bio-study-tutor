# Chargaff's rule

## Answer
**Chargaff's rule**: in any double-stranded DNA, %A = %T and %G = %C (because A always pairs with T and G always pairs with C). Therefore %A + %T + %G + %C = 100%.

**Worked example — 20% T**:
- %T = 20% → %A = 20% (Chargaff)
- %A + %T = 40% → %G + %C = 60% → %G = %C = **30% each**

So: **A = 20%, T = 20%, G = 30%, C = 30%**.
