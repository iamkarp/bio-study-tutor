# Types of DNA mutations

## Answer
A **mutation** is a permanent change in the DNA base sequence. Types covered in class:

| Type | What changes | Effect on protein |
|---|---|---|
| **Point mutation** | A single base is changed | Range from silent to severe |
| **Substitution** | One base is swapped for another | May change one amino acid (missense), create a stop codon (nonsense), or be silent (codon redundancy) |
| **Deletion** | One or more bases are removed | If not a multiple of 3, causes **frameshift** |
| **Insertion** | One or more bases are added | If not a multiple of 3, causes **frameshift** |
| **Frameshift** | Reading frame of codons shifts after the mutation | Usually catastrophic — every amino acid downstream is wrong |

Most mutations occur during DNA replication; some are caused by exposure to **mutagens** (chemicals, UV, radiation).
