# Incomplete dominance

## Answer
**Incomplete dominance**: the heterozygote shows an **intermediate phenotype** between the two homozygotes — neither allele fully masks the other.

Classic example: red snapdragon (R₁R₁) × white snapdragon (R₂R₂) → all **pink** F1 (R₁R₂). Cross two pinks → 1 red : 2 pink : 1 white.

Mechanism: the recessive allele still encodes a non-functional protein. With only one functional copy (heterozygote), the cell makes ~half as much pigment, producing the intermediate color.

**Genotype** is still 1:2:1 (RR : Rr : rr); the difference from complete dominance is that **all three genotypes have distinct phenotypes**, so phenotype ratio = genotype ratio.
