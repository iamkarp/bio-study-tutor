# Codon Redundancy

## Summary
Most amino acids are encoded by **more than one codon** — there are 64 codons but only 20 amino acids. The redundancy is concentrated in the **3rd codon position** (the "wobble" position), where multiple bases often code for the same amino acid. This is why some single-base substitutions are **silent** (no protein change) — the codon still codes for the same amino acid.
