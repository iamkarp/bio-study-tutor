# Mitosis vs. Meiosis

## Comparison

| Feature | Mitosis | Meiosis |
|---|---|---|
| **DNA replications** | 1 (in S phase) | 1 (in S phase before meiosis I) |
| **Cell divisions** | 1 | 2 (meiosis I + II) |
| **Daughter cells** | 2 | 4 |
| **Daughter ploidy** | Diploid (2n) | Haploid (n) |
| **Genetically identical to parent?** | Yes | No |
| **Crossing over?** | No | Yes (prophase I) |
| **Homologues pair up?** | No | Yes (prophase I synapsis) |
| **Anaphase separates** | Sister chromatids | Anaphase I: homologues; Anaphase II: sister chromatids |
| **Function** | Growth, repair, asexual reproduction | Producing gametes for sexual reproduction |
