# Genetic Code

## Summary
The set of rules mapping **mRNA codons to amino acids**. Properties:
- **Triplet** — each codon is 3 nucleotides
- **64 codons total** (4³): 61 specify amino acids, 3 are stop codons (UAA, UAG, UGA)
- **Universal** — nearly the same in all organisms (with rare exceptions)
- **Redundant / degenerate** — most amino acids have multiple codons (see [[concepts/codon-redundancy]])
- **Unambiguous** — each codon specifies one and only one amino acid
- **Start codon AUG** = methionine
- **Read 5' → 3'** on the mRNA
