# Chargaff's Rule

## Summary
In any double-stranded DNA: **%A = %T** and **%G = %C**. So %A + %T + %G + %C = 100%, and %A + %T = 100% − (%G + %C).

This is a direct consequence of [[principles/complementary-base-pairing]] — every A on one strand has a T paired with it on the other, and every G has a C. Erwin Chargaff discovered this empirically in the 1940s; it provided crucial evidence for Watson and Crick's double helix.

## Worked Example
If T = 20%, then A = 20%; G + C must total 60%, so G = C = 30% each.
