# Incomplete Dominance

## Summary
A non-Mendelian inheritance pattern in which the **heterozygote shows an intermediate phenotype** between the two homozygotes. Classic example: **red snapdragon × white snapdragon → pink F1**. With incomplete dominance, **genotype ratio = phenotype ratio** (1:2:1) because each genotype has a unique phenotype.

Mechanistically, the recessive allele still encodes a non-functional protein; with only one functional copy, the cell makes about half as much pigment, producing the blended color.

Distinct from [[inheritance/codominance]] — see [[comparisons/incomplete-vs-codominance|comparison]].
