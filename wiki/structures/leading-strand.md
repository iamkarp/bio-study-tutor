# Leading Strand

## Summary
The strand of newly synthesized DNA that is built **continuously toward the [[structures/replication-fork|replication fork]]**, in the same direction as fork movement. Only **one RNA primer** is needed; [[enzymes/dna-polymerase-iii]] then extends in the 5' → 3' direction without interruption.

Contrast with the [[structures/lagging-strand]], which is built in pieces.
